Competitions

Each semester the Communication Department will be sponsoring visual competitions for you to participate in.
See the current competitions and instructions on how to participate below.

24-Days of Something

Create one refined visual project a day for 24 days. For more information on this see the project requirements at http://ericlybbert.github.io/24-days-of-something/

Intrinsic Motivation

Create one refined visual project that was created for the sole purpose of yourself. This should not be for school, work, etc. For more information on this see the project requirements at www.visualsociety.org/intrinsic-motivation

Motion Project

Create one refined visual project that relays the message of motion. For more information on this see the project requirements at http://ericlybbert.github.io/motion-project/
